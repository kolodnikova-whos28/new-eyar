# New Year Counter API

## О проекте

Данный проект представляет собой серверное приложение, разработанное на языке программирования Go. Приложение реализует REST API, которое вычисляет количество дней, оставшихся до наступления Нового года (1 января следующего календарного года). Ответ возвращается в формате JSON.

**Назначение:** интеграция в системы мониторинга, виджеты обратного отсчета, автоматизированные сценарии оповещения о праздниках.

---

## Функциональные возможности

| Функция | Описание |
|---------|----------|
| Вычисление дней до Нового года | Автоматический расчет на основе системной даты и часового пояса сервера |
| REST API интерфейс | Прием HTTP GET запросов и возврат структурированного JSON ответа |
| Контейнеризация | Поддержка запуска в среде Docker |
| Кроссплатформенность | Работа на операционных системах Windows, Linux, macOS |

---

## Технические требования

Для успешного развертывания и работы приложения необходимо следующее программное обеспечение:

### Локальный запуск
- **Go** версии 1.21 или выше
- **Git** (опционально, для клонирования репозитория)

### Запуск в контейнере
- **Docker** версии 20.10 или выше
